O que é Visual Studio Code?

HTML significa "HiperText Markup Language", ou, em português, "Linguagem de Marcação de Hipertexto, criado pelo britânico Tim Berners-Lee. HTML consiste em uma série de elementos, que você usa para incluir ou envolver diferentes partes do conteúdo para faze-lo aparecer de uma determinada maneira. As tags popdem criar um hiperlink de uma palavra ou imagem para outro lugar, podem colocar palavras em italico, podem aumentar ou diminuir a fonte e assim por diante.
